# PizzaBolt Website

## Overview
PizzaBolt is a responsive, multi-page pizza delivery website built with HTML5 and CSS3. It meets the technical and design requirements for the final project.

## Pages
- **index.html** – Homepage
- **menu.html** – Menu with pizzas and prices
- **gallery.html** – Image gallery
- **about.html** – Company story
- **contact.html** – Form + business hours table

## Features
- Responsive navigation
- External CSS (`style.css`)
- SEO meta tags
- Images on every page
- Contact form and table
- Consistent branding & specialty fonts

## Requirements Met
- 5 pages with 3+ subcategories
- External CSS and consistent layout
- Meta tags, form, table, and images
- Works in all browsers
